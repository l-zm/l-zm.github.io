MacOS Maven的安装与配置

2017-02-28
  • MacOS
  • Maven
  1. 下载
  2. 保存
  3. 设置

下载

https://maven.apache.org/download.cgi

保存

下载好的文件保存在：/Users/lizm/Documents/apache-maven-3.3.3/ (不管在哪里,你开心就好)

设置

打开 终端 , vim ~/.bash_profile

export M2_HOME=/Users/lizm/Documents/apache-maven-3.3.3
export PATH=$PATH:$M2_HOME/bin

保存

source ~/.bash_profile
mvn -v
