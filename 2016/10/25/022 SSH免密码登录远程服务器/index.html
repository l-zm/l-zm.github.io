SSH 免密码登录远程服务器

SSH 免密码登录远程服务器

2016-10-25
  • Linux
  • SSH
  1. 准备
  2. 上传公钥
  3. 修改
  4. Done

准备

本机生成 rsa 秘钥

ssh-keygen -t rsa

上传公钥

scp /Users/Lim/.ssh/id_rsa.pub root@目标服务器IP:/root/.ssh/tmp/

修改

进入：cd /root/.ssh/tmp
修改名字：mv id_rsa.pub authorized_keys
复制到 .ssh 目录：cp /root/.ssh/tmp/authorized_keys /root/.ssh/
修改文件权限：chmod 600 /root/.ssh/tmp/authorized_key (如果不修改，可能会引起 authorized_key 文件被修改,导致文件失效)

Done

直接登录服务器：ssh root@目标服务器IP
